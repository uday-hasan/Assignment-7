Event loop

The works of event loop is how the code will be execute.This event is not like other event handelar.This loop control that which code will execute first and this event loop do this using two steps.One is Microtask queue and another is Macrotask queue.Microtask queue item is syncronus and macrotask queue is asyncronus.At first the microtask queue item move on event loop and then this loop call call stack and then macrotask queue.

Difference between local storage and session storage

Local Storage
  1. It store data for long time.
  2. Local storage data is available in all window after the window has been closed.
Session Storage
  1. It store data for a session
  2. Session storage data is available inside the same browser window till the window is open
